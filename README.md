# Jitendra — App Store

**Live site: https://jitendrajangidcodes-cloud.github.io/app-store/**

A personal catalog of apps built by Jitendra, browsable and installable directly — no store
review, no accounts. Version, download link, and release notes for each listed app are pulled
live from that app's own GitHub Releases, so the site always reflects the latest shipped build.

New apps are never added automatically — an app is only listed here when explicitly added to
`apps.json`.
